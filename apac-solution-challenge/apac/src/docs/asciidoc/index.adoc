== Calendar API

=== [GET] /api/calendar

JWT 인증을 기반으로 사용자의 특정 연/월 목표 달성 현황을 조회합니다.

==== 요청 헤더

[cols="1,2", options="header"]
|===
| 이름 | 설명
| Authorization | `Bearer {JWT}` 형식의 액세스 토큰 (sub에 email 포함)
|===

==== 요청 파라미터

[cols="1,1,2", options="header"]
|===
| 이름 | 타입 | 설명
| year | int | 조회할 연도 (예: `2025`)
| month | int | 조회할 월 (예: `5`)
|===

==== 응답 필드

[cols="1,1,2", options="header"]
|===
| 이름 | 타입 | 설명
| year | int | 응답 연도
| month | int | 응답 월
| days | List | 날짜별 목표 달성 정보 배열
| days[].date | String | 날짜 (`"2025-05-01"` 형식)
| days[].goalAchievedCount | int | 해당 날짜에 달성한 목표 개수
|===
